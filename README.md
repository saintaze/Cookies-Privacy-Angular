# Cookies-Privacy-Angular
A demo angular app showing how user privacy can be controlled and made more secure.

### Features
+ Responsive Design
+ Clean UI

##### Built With
+ HTML5
+ SASS
+ Angular
+ Bootstrap

### Getting Started
clone repo

`cd` into folder and run `npm install`

run `ng serve`
